# 停车场管理系统接线图

## OLED 显示屏

- `STM32 PB8` -> `OLED SCL`
- `STM32 PB9` -> `OLED SDA`
- `STM32 3.3V` -> `OLED VCC`
- `STM32 GND` -> `OLED GND`

## MaixCAM 车牌识别串口

- `STM32 PA9 (USART1_TX)` -> `MaixCAM UART1_RX (A18)`
- `STM32 PA10 (USART1_RX)` <- `MaixCAM UART1_TX (A19)`
- `STM32 GND` -> `MaixCAM GND`

## ESP8266 联网模块

当前稳定方案使用 `ESP8266 SoftwareSerial` 收 STM32 数据：

- `STM32 PB10 (USART3_TX)` -> `ESP8266 GPIO14 (D5)`
- `STM32 GND` -> `ESP8266 GND`
- `STM32 PB11 (USART3_RX)` -> `ESP8266 GPIO12 `

说明：

- `ESP8266` 的 USB 硬件串口保留给日志输出
- `STM32 USART3` 当前波特率使用 `9600`
- `ESP8266` 通过 WiFi 将数据上传到微信云开发

## 红外对管 / 车辆检测

- `STM32 PA0` <- `入口红外 OUT`
- `STM32 PB1` <- `出口红外 OUT`

## 道闸舵机

- `STM32 PA6 (TIM3_CH1 PWM)` -> `舵机信号线`
- `5V 电源` -> `舵机 VCC`
- `STM32 GND` -> `舵机 GND`

## 支付按键

- `STM32 PB12` <- `按键输出`
- `STM32 GND` -> `按键 GND`

## 调试串口（可选）

- `STM32 PA2 (USART2_TX)` -> `USB 转串口 RX`
- `STM32 PA3 (USART2_RX)` <- `USB 转串口 TX`
- `STM32 GND` -> `USB 转串口 GND`
